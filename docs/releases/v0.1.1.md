# cc-clean v0.1.1

本版本聚焦于补齐真正的“全量清理”入口，并继续完善跨平台安全回归测试。

## 本版本包含

- 新增 `clean --purge-all`
  - 清理当前机器上所有已知的 Claude Code 痕迹
  - 覆盖配置、缓存、日志、历史、安装残留、shell / 浏览器 / IDE 集成及系统级项目
- `--purge-all` 内置强制确认机制
  - 即使传入 `-y`
  - 仍必须手动输入 `PURGE-ALL`
- 补充 POSIX / Windows 回归测试
  - 覆盖 `purge-all` 帮助输出
  - 覆盖严格确认流程
  - 覆盖 `ctest` 测试入口

## 重要修复

- 修复 `restore` 阶段多项安全问题：
  - 阻止 `..` 绕过目标路径白名单
  - 阻止通过相对路径逃逸 `backup_dir`
  - 拒绝超长 `manifest.tsv` 行
- 移除 `popen` / `_popen` 调用，收敛为更安全的进程执行方式
- 修复 Linux / Windows 构建兼容性问题
- 修复 Windows 下 `restore` 路径判断与 JetBrains 插件扫描问题
