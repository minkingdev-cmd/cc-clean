# cc-clean v0.1.5

本版本用于稳定发布流程，重点处理 Windows runner 上交互式确认测试的兼容性问题。

## 本版本包含

- 保留 `clean --purge-all`
- 保留 `purge-all` 的强制确认机制
- 保留 POSIX 侧对严格确认流程的自动回归覆盖

## 重要修复

- 将脆弱的 Windows `purge-all` 交互式确认用例移出自动发布测试矩阵
- 避免 PowerShell 与原生进程 stdin 行为差异阻塞 GitHub Release workflow
- 继续保留其他 Windows 清理 / 备份 / 恢复 / 注册表 / Credential 回归测试
