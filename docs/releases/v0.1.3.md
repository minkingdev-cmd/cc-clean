# cc-clean v0.1.3

本版本继续修复 Windows 发布流程中的 `purge-all` 回归测试误判问题。

## 本版本包含

- 保留 `v0.1.1` 引入的 `clean --purge-all`
- 保留 `purge-all` 的强制确认机制
- 修正 Windows 回归测试对原生进程退出码的判断方式

## 重要修复

- Windows `purge-all` 回归测试不再依赖 PowerShell 对原生进程退出码的脆弱采集方式
- 改为直接验证：
  - 错误确认词时目标仍保留
  - 正确确认词时目标已被真正删除
- 使 GitHub Actions Release workflow 能稳定完成 Windows 测试与发布
