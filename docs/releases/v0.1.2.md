# cc-clean v0.1.2

本版本是 `v0.1.1` 的补丁发布，主要修复 Windows 发布流程中的回归测试问题。

## 本版本包含

- 保留 `v0.1.1` 引入的 `clean --purge-all`
- 保留 `purge-all` 的强制确认机制
- 修复 Windows 下 `purge-all` 严格确认回归用例

## 重要修复

- 为 Windows `purge-all` 测试显式提供备份目录，避免测试期间被备份目录交互提示打断
- 使 GitHub Actions Release workflow 能继续完成 Windows 构建、测试与打包
