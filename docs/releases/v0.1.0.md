# cc-clean v0.1.0

`cc-clean` 的首个公开版本，提供一个可独立构建、独立测试、跨平台运行的 **Claude Code 完整卸载与清理工具**。

## 版本亮点

- 面向 **Claude Code 卸载场景** 的独立工具仓库
- 提供 C 语言主实现，支持：
  - macOS
  - Linux
  - Windows
  - x86_64 / aarch64
- 支持清理常见残留内容：
  - 配置目录
  - 缓存目录
  - 日志目录
  - 会话历史
  - 插件目录
  - 凭据与平台相关系统项
- 支持清理前备份与事后恢复
- 支持 JSON 输出，便于自动化集成
- 提供跨平台 GitHub Actions CI

## 本次附带的发布资产

## 推荐使用流程

1. 运行 `check` 查看本机 Claude Code 残留
2. 运行 `clean --backup-dir <目录>` 进行备份与清理
3. 如需回滚，运行 `restore --backup-dir <目录>`

## 工程化内容

- 独立 `CMakeLists.txt`
- 独立 `README.md` / `BUILDING.md`
- CTest 拆分为多个细粒度 case
- 支持按 label 运行测试：
  - `json`
  - `restore`
  - `safety`
  - `dangerous`
  - `manifest`
  - Windows 专项标签
