# cc-clean v0.1.0

`cc-clean` 的首个公开版本，当前标签已对齐到整理后的干净提交历史，并包含稳定可用的跨平台 CI。

## 本版本包含

- 用于**彻底卸载 Claude Code** 的独立跨平台清理工具
- C 语言主实现，支持：
  - macOS
  - Linux
  - Windows
  - x86_64 / aarch64
- 扫描、备份、清理、恢复完整流程
- JSON 输出，便于自动化集成
- GitHub Actions 跨平台构建与测试

## 本次整理后的发布状态

- 标签 `v0.1.0` 已重新对齐到当前干净历史
- Windows CI 测试脚本已稳定
- GitHub Actions 已升级为 `actions/checkout@v5`
- 当前 `main` 分支在 GitHub Actions 上通过：
  - ubuntu-latest
  - macos-latest
  - windows-latest

## 推荐使用流程

1. 运行 `check` 查看本机 Claude Code 残留
2. 运行 `clean --backup-dir <目录>` 进行备份与清理
3. 如需回滚，运行 `restore --backup-dir <目录>`

## 测试覆盖

- CLI 帮助与危险开关
