# cc-clean v0.1.4

本版本继续修复 Windows 发布流程中的 `purge-all` 回归测试兼容性问题。

## 本版本包含

- 保留 `clean --purge-all`
- 保留 `purge-all` 的强制确认机制
- 修复 Windows PowerShell 输出编码导致的断言误判

## 重要修复

- Windows `purge-all` 回归测试不再依赖中文输出文本匹配
- 直接验证：
  - 错误确认词时数据未被误删
  - 正确确认词时目标目录已被真实删除
- 进一步提高 GitHub Actions Release workflow 在 Windows runner 上的稳定性
