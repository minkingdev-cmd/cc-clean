# Security Policy

## Supported Versions

当前优先维护 `main` 与最新发布版本。

## Reporting a Vulnerability

如果你发现与以下方面相关的安全问题，建议尽快提交 issue 或私下联系维护者：

- 卸载路径误删
- 备份 / 恢复越权写入
- Windows 注册表恢复错误
- macOS Keychain / Windows Credential 处理问题
- 非预期的命令执行或平台集成修改

提交安全问题时，建议附带：

- 平台与架构
- 命令行参数
- 最小复现步骤
- 预期结果与实际结果
- 是否会影响现有本地数据
